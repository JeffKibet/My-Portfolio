Flagship project

Menda

A CBT-guided mental wellness web app for the Kenyan market, built with React (Vite), Tailwind CSS, and React Router, with the Anthropic Claude API powering an AI-guided CBT chatbot. Ships a PHQ-9 intake assessment, mood tracking, and a crisis-escalation path, deployed on Vercel. Menda exists because access to mental health support in Kenya is uneven, and the first conversation is often the hardest one to start. It walks someone through a standard clinical intake, gives them a CBT-guided space to talk, tracks their mood over time, and knows when to escalate to real crisis support.

  • React

  • Claude API
  • Vercel
